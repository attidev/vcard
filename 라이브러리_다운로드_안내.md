# 라이브러리 파일 다운로드 안내

인터넷 연결이 불안정하거나 CDN 접근이 차단된 경우, 라이브러리 파일을 직접 다운로드하여 사용할 수 있습니다.

## 다운로드 방법

### 방법 1: 브라우저에서 직접 다운로드

1. **QRCode 라이브러리 다운로드**
   - 브라우저에서 다음 링크를 열기:
     ```
     https://cdn.jsdelivr.net/npm/qrcode@1.5.3/build/qrcode.min.js
     ```
   - 페이지가 열리면 `Ctrl+S` (또는 `Cmd+S`)를 눌러 저장
   - 파일명을 `qrcode.min.js`로 저장
   - 저장 위치: `index.html` 파일과 같은 폴더

2. **JSZip 라이브러리 다운로드**
   - 브라우저에서 다음 링크를 열기:
     ```
     https://cdn.jsdelivr.net/npm/jszip@3.10.1/dist/jszip.min.js
     ```
   - 페이지가 열리면 `Ctrl+S` (또는 `Cmd+S`)를 눌러 저장
   - 파일명을 `jszip.min.js`로 저장
   - 저장 위치: `index.html` 파일과 같은 폴더

### 방법 2: PowerShell 사용 (Windows)

PowerShell을 관리자 권한으로 실행한 후 다음 명령어 실행:

```powershell
# QRCode 라이브러리 다운로드
Invoke-WebRequest -Uri "https://cdn.jsdelivr.net/npm/qrcode@1.5.3/build/qrcode.min.js" -OutFile "qrcode.min.js"

# JSZip 라이브러리 다운로드
Invoke-WebRequest -Uri "https://cdn.jsdelivr.net/npm/jszip@3.10.1/dist/jszip.min.js" -OutFile "jszip.min.js"
```

### 방법 3: 명령 프롬프트 사용 (Windows)

명령 프롬프트(cmd)를 관리자 권한으로 실행한 후:

```cmd
# QRCode 라이브러리 다운로드
curl -o qrcode.min.js https://cdn.jsdelivr.net/npm/qrcode@1.5.3/build/qrcode.min.js

# JSZip 라이브러리 다운로드
curl -o jszip.min.js https://cdn.jsdelivr.net/npm/jszip@3.10.1/dist/jszip.min.js
```

## 파일 구조

다운로드 완료 후 폴더 구조는 다음과 같아야 합니다:

```
프로젝트 폴더/
├── index.html
├── styles.css
├── app.js
├── template.csv
├── qrcode.min.js  ← 다운로드 필요
└── jszip.min.js   ← 다운로드 필요
```

## 확인 방법

라이브러리 파일을 다운로드한 후 `index.html` 파일을 브라우저로 열면 자동으로 로컬 파일을 사용합니다.

CDN이 실패해도 로컬 파일이 있으면 정상적으로 작동합니다!
